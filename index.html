HTML

HTML stands for Hypertext Markup Language. HTML documents form the majority of the content on the web. HTML 5 is the current version we are using on the web.

HTML Tags

Tags tell the browser how the HTML document should look and contain.

HTML Characterics

1. There are two types: Inline and Block. Inline means it the text stays on the same line, while Block places an invisble box around the content.
2. HTML tags can be nested: This means you can have more than one tag inside another tag.
3. HTML tags can also have attributes: This is a property of the HTML tag. This video explains the distinction well.

Example of HTML Tags

All HTML tags have to be contained between "<"opening tag and closing tag "/>" There are exceptions Some tag are call Void tags and they do not need a closing tag.
"b" = bold
"em" = italics
"br" = break
"a" = anchor (used for links)

Always Remember

Computers are stupid and you must pay attention to detail. Misspelled words; unclosed tags; and missed puncutation cn cause your page to break or cause it to be unreadable.
